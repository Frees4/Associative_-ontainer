# Associative container

В файле есть exe файл, который запускает код с тестом из 10 000 элементов и показывает время его выполнения.

Существует две основные реализации Ассоциативного массива: HashMap и Binary Tree. Я выполнил на HashMap

Функции контейнера: 

_get_hash(key) - получает ключ, возвращает hash номер

insert(key, value) - вставляет элемент в контейнер key-int, value-string

remove(key) - удаляет элемент из контейнера

get(key) - возвращает значение(value) элемента по ключу(key)

print() - выводит все элементы контейнера в консоль
